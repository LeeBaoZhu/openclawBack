# 人生算法

《人生算法》是一套将宏大愿景变成可执行路径的思维框架。核心目标：**找到可大规模复制的、具有连续性的"大概率事件"作为内核，收获人生的复利。**

## 快速诊断

当用户请求自我分析时，先问三个问题：
1. 你目前最大的困惑是什么？（定位问题类型）
2. 你希望达成什么目标？（确定方向）
3. 你愿意付出什么代价？（评估资源）

根据回答，匹配到对应的段位或难题。

## A计划：九段心法

A计划是**自我塑造**的过程，把你打造成一辆赛车。

### 上半场（切割钻石，做减法）

#### 初段：闭环

**核心问题**：完成比完美更重要。

**行动指南**：
- 建立最小化闭环：感知 → 认知 → 决策 → 行动
- 用"蚁群算法"思维：多次尝试，逼近最优解
- 培养成长型思维：相信努力能改变结果

**检查清单**：
- 你是否在等"完美时机"？→ 立即开始
- 你是否害怕失败？→ 把失败当作数据采集
- 你是否在意他人评价？→ 给未来一个交代

#### 二段：切换

**核心问题**：在自动驾驶和主动控制模式间自如切换。

**三种攻略**：
1. 把不重要的事交给自动驾驶（如奥巴马只穿灰蓝西装）
2. 自动驾驶后用主动控制复盘（如碧昂斯看演唱会录像）
3. 在大脑中模拟自动驾驶（心理演练）

#### 三段：内控

**核心问题**：认知飞轮的四个内控点。

**四人接力赛**：
| 角色 | 任务 | 特质 |
|------|------|------|
| 感知（情报员） | 获取信息 | 好奇敏感 |
| 认知（分析师） | 评估概率 | 灰度理性 |
| 决策（指挥官） | 作出取舍 | 黑白果断 |
| 行动（战士） | 执行任务 | 疯子行动 |

**提升思考率**：在每个内控点按下暂停键，问"慢，让我想想看"。

#### 四段：重启

**核心问题**：过往不恋，未来不惧。

**两个精神装置**：
1. **外星人视角**：假设外星人接管你的生活，忽视沉没成本，理性评估
2. **阿尔法狗视角**：把当前局面当已知条件，计算最优解

**应用场景**：
- 面对失败 → 外星人视角重启
- 面临选择 → 阿尔法狗视角计算

#### 五段：增长

**核心问题**：找到可持续的增长公式。

**增长黑客三步**：
1. **假设**：建立最小化闭环（单件流，快速试错）
2. **验证**：找到北极星指标（单一关键要素）
3. **执行**：分阶段增长（如Facebook先哈佛→常春藤→全社会）

#### 六段：内核

**核心问题**：找到可复制的最小内核。

**内核特征**：
- 简单：大概率事件、可复制、大规模
- 有系统潜力：能构建护城河

**三步法**：
1. 把握时机（选对时代）
2. 依靠禀赋（发挥天赋）
3. 形成专业（打磨护城河）

**巴菲特原则**：跨1英尺栏杆，而非7英尺。

### 下半场（放大内核，做乘法）

#### 七段：复利

**核心问题**：营造长期的局部垄断。

**关键洞察**：
- 赚钱的公司 ≠ 值钱的公司（未来现金流折现）
- 个人标签 = 你的垄断优势
- 延迟满足 + 持续学习 = 复利基础

#### 八段：愿景

**核心问题**：设计人生导航系统。

**北极星优于地图**：模糊的精确 > 精确的模糊。

**贝佐斯三武器**：
1. 未来十年不变的是什么？
2. 最小化后悔表（80岁回看会后悔吗？）
3. 以终为始战略

#### 九段：涌现

**核心问题**：在自己身上发挥群体智慧。

**系统思维**：
- 每个小行为像一只蚂蚁
- 你的整个人生是蚁群
- 成功要素可能很简单
- 重要的是系统

**黄峥方法**：把自己当资产负债表，分辨资产vs费用。

## B计划：18个人生难题

B计划是**应对外部世界不确定性**，解决人生元问题。

### 难题分类

| 类别 | 难题 | 核心算法 |
|------|------|----------|
| **认知局限** | 片面、狭隘、模糊 | 三个运气旋钮、概率权时间权、量化思维 |
| **风险应对** | 侥幸、宿命、追悔 | 理解随机性、摆脱宿命论、正确复盘 |
| **决策陷阱** | 非理性、冲动、犹豫、武断 | 理性工具、暂停键、决策算法 |
| **社会关系** | 情面、霉运、孤独 | 社交算法、好运配方、孤独价值 |
| **财富风险** | 爆仓、迷信、无知、衰朽、贪婪 | 避免归零、科学思维、终身学习、反脆弱 |

### 重点难题解法

#### 12. 片面 → 三个运气旋钮

**三层思维**：
| 旋钮 | 负责内容 | 关键动作 |
|------|----------|----------|
| 老板旋钮 | 资源层 | 整合资源、改变环境 |
| 教练旋钮 | 赛道层 | 选对赛道、制定战略 |
| 手旋钮 | 专业层 | 全力以赴、执行任务 |

**案例**：洛克菲勒同时转动三个旋钮，从炼油到全产业链整合。

#### 13. 狭隘 → 概率权与时间权

**穷人思维vs富人思维**：
| | 穷人思维 | 富人思维 |
|------|----------|----------|
| 时间权 | 打折甩卖，尽快兑现 | 延迟满足，长期持有 |
| 概率权 | 要确定性，放弃可能性 | 计算期望值，承受风险 |

**经典案例**：两个按钮选择 → 用概率思维打开空间（卖选择权、分赌、做彩票）。

#### 14. 模糊 → 量化思维

**核心原则**：范围比精准更重要，量化是减少不确定性。

**费米估算法**：
1. 列出假设条件
2. 逐步逼向真相
3. 不要求精确

**OKR工具**：
- O（目标）：明确要量化什么
- KR（关键结果）：可检验的动作

## 使用模式

### 模式1：段位诊断

用户请求："帮我看看我卡在哪个段位"

步骤：
1. 询问当前状态（工作、学习、生活）
2. 用九段标准逐一匹配
3. 定位短板段位
4. 给出提升建议

### 模式2：难题破解

用户请求："我面临XX困境"

步骤：
1. 匹配到18难题之一
2. 应用对应算法
3. 给出具体行动建议
4. 设置后续跟踪

### 模式3：愿景拆解

用户请求："我有个目标，但不知道怎么实现"

步骤：
1. 确认愿景（八段）
2. 找到内核（六段）
3. 设计增长路径（五段）
4. 建立闭环系统（初段-三段）
5. 设定复盘节奏

### 模式4：复盘引导

用户请求："帮我复盘一下"

步骤：
1. 回顾目标
2. 评估结果
3. 分析差异原因
4. 调整下一步
5. 重启下一个飞轮
