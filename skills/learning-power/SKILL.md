# 学习力技能

提升学习效率的五项修炼与核心工具，基于刘澜《学习力30讲》方法论。

## 核心框架

### 学习力公式

```
学习力 = 反学习 + 参考答案思维 + 聚焦 + 模式化学习 + 深层迁移
```

### 有效学习的标准

```
有效学习 → "听想变用"四维度产生长期变化
```

---

## 五项修炼

### 第一项：反学习

**核心理念**：成年人首先要清除错误观念，而非增加新知识。

**高阶反学习**：清除心智模式，而非单个知识点

**10个学习坏习惯**（需清除）：
1. 标准答案思维方式 — 总是寻找标准答案
2. 被动式学习 — 不划重点、不记笔记
3. 害怕出错 — 认为出错是学得不好
4. 追求全面发展 — 弥补短板而非发挥擅长
5. 缺乏学习规划
6. 习惯一个人学习
7. 期待专家提供答案
8. 碎片化学习 — 知识之间没有联系
9. 学而不用
10. 注重结果而非过程

**反学习方法**：用加法做减法 — 建立新习惯替代旧习惯

---

### 第二项：参考答案思维方式

**核心理念**：重要问题没有标准答案，只有参考答案。

**口诀**："看情况"

**三个原则**：
1. **真理是看情况的** — 三种真理论：符合论、融贯论、实用论
2. **实用是看情况的** — 好理论的三个标准：一般性、准确性、简单性
3. **学习实用论** — 不仅要发现"正确"的答案，更要发现"有用"的答案

**五个策略**：

| 策略 | 方法 | 示例 |
|------|------|------|
| 洞见分类 | 主观、理想、本质的分类 | 团队分三种：棒球队、橄榄球队、篮球队 |
| 标准参考答案 | 在绝大多数情况下管用的答案 | "看情况" → 分类 → 具体情况 |
| 互动式学习 | 不同参考答案之间的碰撞 | 守破离、事前建构、单人互动 |
| 反过来想 | 对立式、互补式、因果式 | 末位淘汰 → 首位淘汰？ |
| 注重过程 | 重点学习得出答案的过程 | 观摩高手的不完美过程 |

**合理质疑三策略**：
1. 问"样本有代表性吗？"
2. 警惕平均数
3. 注意非线性（好东西并非多多益善）

### 第三项：聚焦

**核心理念**：学得少而深，而非多而浅。

**学习公式**：
```
学习 = 我知道 + 我不知道
```
（用已有知识去吸收新知识，产生马太效应）

**四个规则**：

| 规则 | 方法 | 关键点 |
|------|------|--------|
| 聚焦不变 | 学习不变的规律和经典 | 向导之书 → 经典之书 → 反复读 |
| 聚焦实用 | 围绕人生三大课题学习 | 工作、人际关系、个人成长 |
| 聚焦专题 | "大盗式学习法" | 找准问题 → 读100本书 → 提炼自己的体系 |
| 聚焦样本 | "苏东坡式学习法" | 每次从一个角度深入学习一个样本 |

**小偷式学习法** vs **大盗式学习法**：
- 小偷式：广泛学习、目标明确、快速收获
- 大盗式：建立完整体系后再用小偷式补充

---

### 第四项：模式化学习

**核心理念**：学习事物的深层结构（模式），而非表面特征（碎片）。

**模式化学习金字塔**（从低到高）：

| 层次 | 名称 | 特征 |
|------|------|------|
| 1 | 零学习 | 什么都没学到 |
| 2 | 单点碎片化学习 | 学到一个知识点 |
| 3 | 多点碎片化学习 | 学到多个知识点，无联系 |
| 4 | 模式化学习 | 发现知识点之间的关系 |
| 5 | 模式迁移 | 把模式应用到其他领域 |

**特点**：先难后易 — 掌握模式后，碎片知识更容易学习

**四个方法**：

| 方法 | 操作 |
|------|------|
| 用模式吸收碎片 | 中距离学习模式 → 辨别碎片 → 寻找碎片 → 安放碎片 |
| 用模式分析碎片 | 推测 → 检验 → 推广 |
| 用模式比较模式 | 存同 → 求异 → 取舍 |
| 用模式联系模式 | 发现不同模式背后的高阶模式 |

---

### 第五项：深层迁移

**核心理念**：把一个领域的模式应用到另一个领域。

**迁移分类**：
- 表层迁移：直接迁移碎片（往往是错的）
- 深层迁移：先提炼模式，再迁移模式（真正有效）

**核心工具：四问学习法**

```
听 → 我听到什么？（听清楚、听明白）
想 → 我想到什么？（想出背后的模式）
变 → 我变成什么？（调整为适合自己的）
用 → 我用到哪里？（明确具体应用场景）
```

**四大误区与克服技巧**：

| 误区 | 技巧 |
|------|------|
| 听得太单调 | 诠释学四个问题：究竟说什么？想要说什么？能够说什么？应该说什么？ |
| 想得不深刻 | 从三个方向想：其他碎片、自己的问题、已有模式 |
| 变得不灵活 | 换词造句：移植、放大、缩小、颠倒 |
| 用得太宽泛 | 执行意图："如果...就..."句式 |

## 快速诊断

当用户询问学习问题时，按以下流程诊断：

```
1. 检查学习目标 → 是否聚焦？（修炼三）
2. 检查学习方法 → 是否模式化学习？（修炼四）
3. 检查应用情况 → 是否深层迁移？（修炼五）
4. 检查底层思维 → 是否有标准答案思维？（修炼二）
5. 检查已有习惯 → 是否有学习坏习惯？（修炼一）
```

## 使用场景

### 场景1：学习效率低
- 诊断：可能在学习碎片而非模式
- 建议：用模式化学习（修炼四）
- 工具：模式化学习金字塔

### 场景2：学了不会用
- 诊断：表层迁移，没有提炼模式
- 建议：用四问学习法进行深层迁移
- 工具：听想变用四步骤

### 场景3：学了很多但没进步
- 诊断：没有聚焦，学得太多太浅
- 建议：聚焦一个专题深入学习
- 工具：大盗式学习法

### 场景4：不知道学什么
- 诊断：缺乏人生三大课题的聚焦
- 建议：围绕工作、人际关系、个人成长学习
- 工具：小偷式学习法

### 场景5：害怕学错
- 诊断：标准答案思维方式
- 建议：建立参考答案思维方式
- 工具："看情况"三字口诀

## 关键洞见

1. **学习力的本质**：不是学得多，而是学得深、用得出

2. **成年人与儿童的区别**：儿童需要加法式学习，成年人需要减法式反学习

3. **专家与新手的核心区别**：专家看到模式，新手看到碎片

4. **最有效的学习公式**：学习 = 我知道 + 我不知道（已有知识吸收新知识）

5. **四问学习法是核心工具**：所有学习高手都在无意识地使用"听想变用"
