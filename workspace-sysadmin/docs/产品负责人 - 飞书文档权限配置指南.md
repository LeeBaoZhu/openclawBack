# 📌 产品负责人 - 飞书文档权限配置指南

_配置此权限后，AI 助手可以帮你读写飞书文档_

---

## 🔧 配置步骤

### 第一步：进入飞书开放平台

1. 访问：https://open.feishu.cn/
2. 登录你的飞书账号
3. 进入「企业自建应用」

### 第二步：选择应用

找到并点击「产品负责人」应用

### 第三步：添加权限

1. 点击左侧菜单「权限管理」
2. 点击「批量导入」
3. 粘贴以下权限配置：

```json
{
  "scopes": {
    "tenant": [
      "docs:docs:readonly",
      "docs:docs:write",
      "docs:files:readonly",
      "docs:files:write"
    ]
  }
}
```

4. 点击「确定」保存

### 第四步：发布应用

1. 点击「版本管理与发布」
2. 创建新版本
3. 提交发布（企业应用通常自动审批）

---

## ✅ 权限说明

| 权限 | 用途 |
|------|------|
| `docs:docs:readonly` | 读取飞书文档内容 |
| `docs:docs:write` | 创建/编辑飞书文档 |
| `docs:files:readonly` | 读取云空间文件 |
| `docs:files:write` | 上传/写入云空间文件 |

---

## 📌 配置后的使用方式

配置完成后，AI 助手可以帮你：

- ✅ 编辑现有文档内容
- ✅ 读取文档内容
- ✅ 上传文件到云空间

## ⚠️ 注意事项

1. **配置后需要重启 Gateway** - 权限配置完成后，需要重启 OpenClaw Gateway 才能生效
2. **不需要配置 MCP** - OpenClaw 已内置 `feishu_doc` 技能，只需配置飞书权限即可
3. **权限最小化原则** - 只配置必要的权限，不要过度授权

---

## 🆘 遇到问题？

如果配置后仍无法使用，请检查：

1. 权限是否已正确添加并保存
2. 应用是否已发布
3. Gateway 是否已重启
