# emotion-coaching - 情绪管理技能

## 描述

基于脑科学和儿童发展心理学，帮助家长理解孩子情绪发展规律，掌握处理孩子发脾气、情绪崩溃的方法，培养孩子情绪调节能力。

## 核心原理

**情绪脑发育**：
- 孩子情绪调节能力随大脑发育逐渐成熟
- 前额叶（理智脑）20 岁后才完全发育成熟
- 情绪爆发是学习情绪调节的过程，不是"不听话"

**共情优先**：
- 先处理情绪，再处理事情
- 命名情绪帮助孩子理解自己的感受
- 提供选择让孩子有掌控感

## 触发场景

用户提到以下话题时激活此技能：
- 孩子发脾气/哭闹
- 孩子情绪崩溃
- 孩子打人/咬人
- 孩子说"不"/唱反调
- 孩子公共场合闹情绪
- 如何安抚受惊的孩子
- 孩子情绪管理差
- "可怕的两岁"叛逆期

## 三步搞定发脾气的娃

### 1. 共情：理解并命名情绪

**原则**：先接纳情绪，再引导行为

**错误做法**：
- ❌ "不许哭！"
- ❌ "这有什么好哭的"
- ❌ "再哭就不要你了"
- ❌ 讲道理（情绪上来时听不进去）

**正确做法**：
- ✅ 蹲下来，平视孩子
- ✅ 用语言命名孩子的情绪
- ✅ 表示理解孩子的感受

**话术示例**：
- "你现在很生气，对不对？"
- "妈妈知道你很伤心，因为……"
- "你很想继续玩，现在要回家了，你很失望"
- "被吓到了是不是？爸爸在这里"

**效果**：孩子感觉被理解，情绪开始平复

### 2. 设立边界：情绪可以，行为不可以

**原则**：接纳情绪，但限制不当行为

**错误做法**：
- ❌ 完全放任（"随便他哭"）
- ❌ 体罚或威胁
- ❌ 妥协（"好了好了给你买"）

**正确做法**：
- ✅ 明确什么行为不可以
- ✅ 提供可接受的替代方式
- ✅ 保持冷静，不被孩子情绪带动

**话术示例**：
- "生气是可以的，但打人是不可以的"
- "你可以哭，也可以跺脚，但不能扔东西"
- "我们可以在这里安静一会儿，等你好一点再走"
- "你可以选择现在走，或者再玩 5 分钟走"

**效果**：孩子学习适当表达情绪的方式

### 3. 引导：提供解决方案

**原则**：情绪平复后，引导解决问题

**错误做法**：
- ❌ 事后说教（"早跟你说了……"）
- ❌ 翻旧账
- ❌ 要求立即道歉

**正确做法**：
- ✅ 等孩子完全平静后再讨论
- ✅ 一起想办法解决问题
- ✅ 肯定孩子的进步

**话术示例**：
- "现在你好一点了吗？我们想想怎么办"
- "下次遇到这种情况，我们可以怎么做？"
- "你今天虽然很生气，但没有打人，这就是进步"
- "我们来想想有什么办法可以让你不那么生气"

**效果**：孩子学习问题解决能力

## 常见场景应用

### 场景 1：公共场合哭闹
1. 共情："我知道你现在很想要那个玩具"
2. 边界："但我们今天不能买，你可以哭一会儿"
3. 引导："我们把它记在愿望清单上好不好"
4. 必要时带离现场，等平静再回来

### 场景 2：被鞭炮声吓到
1. 共情："声音太大了，吓到你了是不是"
2. 安抚："爸爸抱着你，没事的"
3. 解释："这是鞭炮声，过年时会放，声音大但不危险"
4. 逐步脱敏：远一点看，慢慢靠近

### 场景 3："可怕的两岁"唱反调
1. 理解：这是自主意识发展期，正常现象
2. 提供选择："你想穿红色还是蓝色"
3. 避免权力斗争：不硬碰硬
4. 保持规律：作息规律减少情绪波动

### 场景 4：入园哭闹
1. 共情："你舍不得妈妈，对不对"
2. 建立仪式感：固定告别方式（拥抱、击掌）
3. 明确告知："妈妈下午 X 点来接你"
4. 遵守承诺：准时出现，建立信任

## 情绪类型及应对

| 情绪 | 可能原因 | 应对方法 |
|------|----------|----------|
| 生气 | 需求未被满足、被阻止 | 共情 + 命名 + 提供选择 |
| 伤心 | 失去、分离、挫折 | 拥抱 + 陪伴 + 安慰 |
| 害怕 | 陌生、巨响、黑暗 | 保护 + 解释 + 逐步脱敏 |
| 失望 | 期望落空 | 理解 + 替代方案 |
| 焦虑 | 变化、不确定性 | 预告 + 规律 + 安抚物 |

## 家长情绪自我修养

**原则**：育儿育己，先管理好自己的情绪

**具体做法**：
- 意识到自己情绪上来时，先深呼吸
- 暂时离开现场（确保孩子安全）
- 告诉自己"这是孩子学习情绪的过程"
- 事后复盘：什么触发我的情绪

**话术示例**：
- "妈妈现在也有点生气，需要冷静一下"
- "我们都需要安静一会儿，等下再聊"

**效果**：给孩子树立情绪管理的榜样

## 常见误区

| 误区 | 正确做法 |
|------|----------|
| 不许哭/不许生气 | 情绪可以，行为有限制 |
| 讲道理（情绪上来时） | 先共情，平静后再讲 |
| 威胁/体罚 | 设立边界 + 提供选择 |
| 立即妥协 | 温和而坚定地坚持 |
| 事后说教 | 引导解决问题 |
| 贴标签（"你就是爱哭"） | 描述行为，不贴标签 |

## 核心理念

> 情绪爆发正常，是学习情绪调节的过程。

> 先处理情绪，再处理事情。

> 育儿育己：你通过教养孩子，自己也变得越来越会管理情绪了。

## 输出格式

回答应包含：
1. **情绪理解** — 孩子情绪发展的科学解释
2. **具体方法** — 三步法（共情、边界、引导）
3. **话术示例** — 可直接使用的沟通话术
4. **家长修养** — 家长自身情绪管理建议

## 边界

- 不诊断心理疾病（建议就医）
- 不制造焦虑（情绪爆发是正常现象）
- 尊重个体差异（气质不同）
- 拿不准的问题标记「待确认」
