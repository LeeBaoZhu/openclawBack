# 育儿技能索引

基于《Dr.魏的家庭教育宝典》（教育宝典.md）创建的育儿技能列表。

## 已创建技能

### 1. 好奇心保护（curiosity-protect）
**核心内容**：保护孩子的好奇心、探索精神和灵活思维能力
**三大方法**：
- 鼓励探索（大自然、博物馆、游乐场）
- 家庭探索引导（开放式玩具、熟悉场景新玩法）
- 避免负面评判（不否定孩子的发现）
**触发场景**：孩子好奇心强/弱、爱问为什么、探索行为、参观博物馆等

### 2. 成长型心智培养（growth-mindset）
**核心内容**：正确表扬孩子，培养成长型心智模式
**三招心法**：
- 表扬努力，而不是聪明
- 表扬过程，而不是结果
- 表扬成长，而不是表现
**触发场景**：如何表扬孩子、孩子求表扬、害怕失败、躲避挑战等

### 3. 专注力培养（focus-training）
**核心内容**：通过心流三原则引导孩子自发专注
**三步骤**：
- 设立明确的目标（任务具体化）
- 难度与能力匹配（跳一跳够得着）
- 及时和全面的反馈
**触发场景**：注意力不集中、做作业磨蹭、学琴不专注等

### 4. 电子设备管理（screen-time-management）
**核心内容**：科学管理孩子使用电子设备
**三大限制方法**：
- 限定时间（每次 + 每天总时间）
- 限定场景（吃饭、睡觉前、房间）
- 提供替代方案（运动、社交、桌面游戏）
**产品选择三原则**：互动性强、创造性强、生活化
**触发场景**：孩子能玩手机吗、沉迷电子设备、如何选择教育产品等

### 5. 科学思维培养（scientific-thinking）
**核心内容**：培养孩子科学思维、探究精神和"问为什么"的能力
**核心方法**：
- 逛博物馆：重点不是知识点，而是思考
- 带着问题逛博物馆
- 尽量去能"动"的博物馆
- 日常保护"问为什么"
**触发场景**：逛博物馆、培养科学思维、孩子爱问为什么等

### 6. 情绪管理（emotion-coaching）
**核心内容**：处理孩子发脾气、情绪崩溃，培养情绪调节能力
**三步法**：
- 共情：理解并命名情绪
- 设立边界：情绪可以，行为不可以
- 引导：提供解决方案
**触发场景**：孩子发脾气、哭闹、公共场合闹情绪、"可怕的两岁"等

### 7. 社交能力培养（social-skills）
**核心内容**：培养孩子分享、合作、解决冲突、同理心等社交技能
**核心技能**：
- 学会分享（尊重物权 + 引导轮流）
- 解决冲突（不急于介入，引导自主解决）
- 培养同理心（理解他人感受）
- 假装游戏（过家家是社交练习）
**触发场景**：不分享、抢玩具、打人/被打、不合群、入园适应等

### 8. 数学启蒙（math-启蒙）
**核心内容**：科学进行 0-6 岁数学启蒙，培养数感和数学思维
**核心内容**：
- 培养数感（第一步，比计算重要）
- 推理能力（分类、排序、模式识别）
- 空间认知（积木、拼图、方向词汇）
- 数学语言（生活中自然使用）
**触发场景**：如何数学启蒙、学数数、要不要上早教班、培养数感等

### 9. 阅读能力培养（reading-development）
**核心内容**：培养阅读兴趣和能力，包括亲子阅读、识字、绘本选择
**核心方法**：
- 互动阅读法（提问、讨论、表演）
- 把绘本当舞台剧（分角色演绎）
- 识字方法（生活中自然识字，不急于过早）
- 绘本选择（年龄匹配、多样性）
**触发场景**：亲子阅读、孩子不爱看书、识字太早好不好、选择绘本等

### 10. 记忆力提升（memory-boost）
**核心内容**：理解和提升孩子记忆力，包括工作记忆训练和记忆策略
**核心内容**：
- 工作记忆游戏（数字、指令、视觉）
- 记忆策略（理解优于死记、组块化、多感官、提取练习）
- 睡眠与记忆（保证充足睡眠）
- 多元思维培养（不贴标签）
**触发场景**：记性差、学了就忘、如何提升记忆力、记忆方法等

## 技能使用规范

### 激活方式
技能会在用户提到相关话题时自动激活，提供：
1. **科学依据** — 简要说明脑科学/心理学原理
2. **具体方法** — 可操作的核心方法
3. **话术示例** — 可直接使用的沟通话术（❌错误 vs ✅正确）
4. **注意事项** — 常见误区提醒

### 工作原则
- 科学依据优先，不传播育儿谣言
- 分龄分段，建议匹配孩子当前年龄（女儿快 3 岁）
- 尊重个体差异，不制造焦虑
- 实用可操作，不是空洞理论
- 拿不准的问题标记「待确认」，涉及医疗建议就医

### 边界
- 不提供医疗诊断（疑似问题建议就医）
- 不传播未经科学验证的方法
- 不制造焦虑（如"XX 岁不会这个就晚了"）
- 尊重家庭养育方式差异
