# 常见健康问题与疾病参考

本目录用于存放**非诊断性**的疾病/健康问题参考，供健康顾问在提供「疾病相关建议」时查阅。内容仅用于生活管理、就医时机、日常注意、复诊准备等建议，不用于诊断或治疗。

## 使用原则

- **仅作参考** — 顾问可据此给出生活与就医建议，不可替代医生诊断与处方
- **来源可靠** — 优先采用医学共识、指南与权威机构资料，并随使用补充
- **及时就医** — 任何急症或不确定情况，一律建议用户就医

## 文件约定

- 每个疾病或健康主题一个文件，如 `高血压.md`、`2型糖尿病.md`、`感冒与流感.md`
- 内容结构建议包含（可根据需要增删）：
  - **概述** — 简要说明（非诊断定义）
  - **就医时机与科室** — 何时应就诊、建议科室
  - **生活与饮食注意** — 通用性建议
  - **运动与活动** — 安全范围内的建议
  - **日常监测** — 可自我记录的指标与方式
  - **复诊与随访** — 带齐资料、可问医生的问题
  - **警示症状** — 出现时建议立即就医的症状
- 文件末尾注明「以上为健康管理参考，具体诊断与治疗请遵医嘱」

## 参考示例

参见同目录下的示例文件（如 `高血压-参考.md`），便于后续按相同结构补充其他疾病或健康问题。
