# 科学减肥技能 (Scientific Weight Loss)

## 触发条件
- 用户询问减肥、减脂、瘦身相关问题
- 用户需要减肥方案或建议
- 用户询问减肥方法的有效性

---

## 核心框架

### 一、三体目标（科学减肥的终点）

减肥不能只看体重，必须同时关注三个维度：

| 维度 | 指标 | 标准（亚洲人） |
|------|------|----------------|
| **体重** | BMI | 男性 18.5-24，女性 18.5-23 |
| **体脂** | 体脂率 | 男性 15%-18%，女性 20%-25% |
| **体型** | 腰围/腰臀比 | 男性腰围<85cm，腰臀比<0.9；女性腰围<80cm，腰臀比<0.8 |

**核心原则**：为健康而生活，捎带瘦身。

---

### 二、第一性原理：能量守恒

人体是能量守恒系统，减肥必须打造能量缺口。

**增加消耗（出水）**：
- 基础代谢 → 很难改变
- 消化吸收 → 无法控制
- 体温维持 → 不现实
- 睡眠减少 → 反而增肥
- **运动和日常活动 → 唯一可控！**

**减少摄入（进水）**：
- 碳水化合物 → 适当减少，多吃纤维素
- 脂肪 → 严格控制（吸收率96%）
- 蛋白质 → 适当补充（每公斤体重1-1.5g）

**口诀**：合理碳水、严控脂肪、补充蛋白。

---

### 三、唯一正路：生活方式调整

**生活方式三角形**：饮食 + 运动 + 心理因素，缺一不可。

**两个认知误区**：
1. 生活方式调整不是"慢"，而是性价比最高
2. 生活方式调整不难，21天养成一个新习惯

## 饮食方案

### 能量缺口

**目标缺口**：每天 500-750 大卡

**三步打造能量缺口**：
1. **看标签** — 学会看营养成分表
2. **饮食拍照** — 记录三天（工作日、周末、特殊日）
3. **制定餐单** — 从照片中减掉500-750大卡

### 挑选食物四准则

| 准则 | 说明 |
|------|------|
| **加工少** | 新鲜食材，简单烹饪；火锅 > 炒菜 > 油煎 > 油炸 |
| **食物完整** | 蔬果不削皮、不榨汁；谷物选全麦、糙米 |
| **脂肪不饱和** | 植物油、坚果、深海鱼；坚决不吃反式脂肪酸 |
| **蛋白质纯** | 鱼虾 > 鸡鸭 > 牛羊猪；推荐豆制品 |

### 餐盘法则（2-1-1）

```
┌─────────────┬─────────────┐
│             │             │
│   蔬菜水果   │    主食     │
│   （一半）   │   （1/4）   │
│             │             │
├─────────────┼─────────────┤
│             │             │
│   蔬菜水果   │  瘦肉/豆制  │
│   （一半）   │  品（1/4）  │
│             │             │
└─────────────┴─────────────┘
```

- **一半**：蔬菜水果（约半斤蔬菜 + 4两水果）
- **1/4**：全谷物主食（约3两）
- **1/4**：瘦肉/豆制品（约2两瘦肉 + 1两豆腐）

### 饮食方案对比

| 方案 | 特点 | 效果 | 推荐度 |
|------|------|------|--------|
| 极低能量饮食 | <800大卡/天 | 短期快，长期反弹 | ❌ 不推荐 |
| 生酮饮食 | 碳水<10%，脂肪>70% | 3个月减5-6kg，副作用多 | ⚠️ 谨慎使用 |
| 低碳水饮食 | 碳水10%-25% | 6个月减8kg+ | ✅ 短期可用 |
| 低脂饮食 | 脂肪<30% | 1年减8kg，难坚持 | ⚠️ 需毅力 |
| **地中海饮食** | 均衡，碳水50%-60% | 长期有效不反弹 | ✅✅ 最推荐 |

## 运动方案

### 燃脂运动三要素

1. **有氧** — 快走、慢跑、游泳、骑车
2. **强度合适** — 最佳燃脂心率区间
3. **时长足够** — 至少30分钟

### 最佳燃脂心率

```
最佳燃脂心率 = (220 - 年龄 - 静息心率) × 40%-60% + 静息心率
```

### 运动建议

| 类型 | 建议 | 说明 |
|------|------|------|
| 有氧运动 | 每周150分钟以上 | 中等强度 |
| HIIT | 25分钟 ≈ 41分钟中强度 | 高强度，需注意安全 |
| **快走** | 每天30分钟 | **减肥成功率最高！** |
| 力量训练 | 配合有氧 | 塑形，改善腰臀比 |

**塑形重点**：有氧减内脏脂肪 + 力量训练塑形

## 心理因素管理

### 压力管理三招

1. **转变思维** — 制定短期小目标，完成后奖励自己
2. **自我监测** — 每天称体重，给饮食拍照并备注想法
3. **舒缓宣泄** — 瑜伽、冥想、正念、太极

### 睡眠管理三要点

| 要点 | 标准 | 说明 |
|------|------|------|
| 时长 | 6-8小时 | 睡眠消耗400-500大卡 |
| 深度 | 有深度睡眠 | 瘦素增加，饥饿感减少 |
| 规律 | 相对固定 | 熬夜不可怕，怕的是不规律 |

**睡眠要点**：身体累一点，大脑松一点，睡前4小时不吃东西。

- 朋友的朋友的朋友变胖 → 你变胖概率 10%

## 医学干预

### 减肥药物

**适用人群**：
- BMI ≥ 27 且合并肥胖相关疾病
- BMI ≥ 30

**中国唯一批准的减肥药**：奥利司他
- 效果：1年减重约2.6kg
- 副作用：脂肪性腹泻

**注意**：药物必须配合生活方式调整，且需长期服用。

### 减肥手术

**适用人群**：
- BMI ≥ 40
- BMI ≥ 35 且合并严重糖尿病

**效果**：12年后，93%的人仍比手术前轻10%

**注意**：抽脂术不减肥，只塑形。

### 轻断食

**医学定义**：
- 每周2-3天
- 断食日摄入 500-700 大卡
- 在 6-8 小时内吃完

**食物选择**：
1. 升糖指数低的食物
2. 保证每公斤体重 ≥1g 蛋白质
3. 补充维生素和纤维素

**渐进方案**（6个月）：
- 第1个月：每周1天，1000大卡，10小时进食窗口
- 第2个月：每周2天，1000大卡，8小时进食窗口
- 第3个月：每周2天，750大卡，6小时进食窗口
- 第4-6个月：每周2天，500大卡，6小时进食窗口

## 代餐

### 效果
- 1年减重约7.7kg
- 长期有效，不反弹
- 安全，营养更均衡

### 选择标准

1. **含有能量**：150-350大卡
2. **能量搭配合理**：碳水<40%，蛋白质20%-30%，脂肪<30%
3. **维生素和微量元素充足**

### 使用要点
- 代餐必须代替正餐，吃完不能再吃其他东西
- 必须配合饮食调整和运动
- 养成健康生活方式前，代餐不能停

## 青少年肥胖

### 重要认知

⚠️ **没有婴儿肥、青春肥，小时候胖就是真的胖！**

- 肥胖儿童成年后继续肥胖概率：42%-63%
- 肥胖青少年成年后继续肥胖概率：70%-80%

### 十招避免孩子肥胖

1. 全家统一认知
2. 父母管好体重再生娃
3. 新生儿体重 2.5-4 公斤
4. 母乳喂养至少到1周岁
5. 果汁饮料要封顶（6岁后每天最多240ml）
6. 三餐让孩子自己吃，没吃饱不给零食
7. 适度寒冷，少穿一件衣服
8. 让孩子参与做饭
9. 限制电子产品，每天≤1小时
10. 全家11点前熄灯，孩子睡足10小时

## 执行流程

### 减肥咨询流程

```
1. 评估是否需要减肥
   ├─ 计算 BMI、体脂率、腰围
   └─ 对照三体目标标准

2. 分析生活方式
   ├─ 绘制体重变化图
   ├─ 找出关键事件和问题
   └─ 了解饮食、运动、睡眠、压力

3. 设定目标
   ├─ 6个月减体重的 5%-10%
   └─ BMI>35 可减 10%-15%

4. 制定方案
   ├─ 饮食：打造能量缺口，2-1-1餐盘
   ├─ 运动：每周150分钟中强度有氧
   └─ 心理：压力管理，睡眠优化

5. 构建社交圈
   ├─ 家人参与
   ├─ 朋友陪伴
   └─ 社群支持

6. 持续追踪
   ├─ 每周称体重
   ├─ 每周运动≥200分钟（维持期）
   └─ 定期复盘调整
```
