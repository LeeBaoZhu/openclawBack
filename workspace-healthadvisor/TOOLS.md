## 健康数据库

### BMI 计算公式
```
BMI = 体重 (kg) / 身高² (m²)

正常范围：18.5 - 24.9
偏瘦：< 18.5
偏胖：25 - 29.9
肥胖：≥ 30
```

### 每日热量需求
```
基础代谢率 (BMR):
男性：BMR = 88.36 + (13.4 × 体重 kg) + (4.8 × 身高 cm) - (5.7 × 年龄)
女性：BMR = 447.6 + (9.2 × 体重 kg) + (3.1 × 身高 cm) - (4.3 × 年龄)

每日热量需求 = BMR × 活动系数
活动系数:
- 久坐不动：1.2
- 轻度活动：1.375
- 中度活动：1.55
- 高度活动：1.725
```

### 营养素推荐摄入量
```
蛋白质：0.8-1.2 克/公斤体重
碳水化合物：占总热量 50-60%
脂肪：占总热量 20-30%
膳食纤维：25-30 克/天
水：30-40 毫升/公斤体重
```

### 运动心率区间
```
最大心率 = 220 - 年龄

燃脂区间：最大心率的 60-70%
有氧区间：最大心率的 70-80%
无氧区间：最大心率的 80-90%
```

## 服务流程

### 健康咨询标准流程
1. **健康状况了解** — 询问用户健康状况
2. **问题分析** — 分析健康问题的性质
3. **信息检索** — 查找科学的健康信息
4. **方案制定** — 制定个性化的健康方案
5. **详细指导** — 提供完整的健康指导
6. **效果跟踪** — 询问执行效果，调整方案

### 运动指导标准流程
1. **体质评估** — 了解用户体质状况
2. **目标设定** — 设定合理的运动目标
3. **计划制定** — 制定运动计划
4. **动作指导** — 指导正确的运动姿势
5. **进度跟踪** — 跟踪运动进展
6. **方案调整** — 根据进展调整计划

### 营养建议标准流程
1. **饮食分析** — 了解用户饮食习惯
2. **营养评估** — 评估营养状况
3. **需求计算** — 计算营养需求
4. **方案制定** — 制定饮食方案
5. **食谱推荐** — 推荐健康食谱
6. **效果追踪** — 追踪营养改善情况
