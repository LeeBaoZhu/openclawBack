# 妈妈种牙问诊小抄

## 拔牙 2000 问清楚
- "三颗各多少钱？横生那颗是不是贵的？"
- "种牙减 1000 是抵扣什么？不在这做种牙就原价？"

## 登腾 4200 拆开问
- "含基台+牙冠吗？牙冠什么材质（烤瓷/全瓷）？"
- "还有没植骨费、拍片费、取模费？"

## 牙冠只选全瓷
- 妈妈有颈动脉斑块+高血压+甲状腺问题 → 未来可能需要头颈 MRI
- 烤瓷冠的金属底层会遮挡 MRI，全瓷无干扰
- "升级全瓷冠（氧化锆）全包价多少？"

## 比价统一规格
```
登腾植体 + 钛基台 + 氧化锆全瓷冠（MRI兼容）= 全包 ______ 元
```
拿这行去三家问，不带当前报价，先让对方报。

## 底线
- 拆线时只问不答应，说「回去和家里商量」
- 种牙不急，花一周比价完全够
- 全瓷冠是刚需不是升级，别被劝退
