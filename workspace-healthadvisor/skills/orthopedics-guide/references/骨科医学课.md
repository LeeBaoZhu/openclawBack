# 发刊词｜生命在于运动，运动就归骨科

## 运动系统的重要性

如果运动系统出问题，人动不了，所有的人生可能都会塌陷。

腰椎间盘突出 → 腰疼到没法坐着，出门都费劲
肩周炎 → 打开手机、点个赞都干不了

**保护运动系统 = 保障生活品质**

## 三个认识不足

1. 经常做错事伤害自己（久坐、低头玩手机）
2. 对身体信号置之不理（背疼不重视）
3. 以为医学没办法（不知道现代骨科已经很先进）

# 01｜运动系统：为什么骨科不只管骨头？

## 运动系统四要素

| 要素 | 作用 | 比喻 |
|------|------|------|
| 骨骼 | 支撑身体 | 汽车支架 |
| 肌肉 | 提供动力 | 汽车引擎 |
| 软骨 | 缓冲减压 | 汽车轮胎 |
| 韧带 | 连接固定 | 汽车螺丝 |

**四要素缺一不可。**

## 挂号判断

骨科主要处理**躯干骨和四肢骨**，以及相关的肌肉、软骨、韧带。

- ✅ 大腿肌肉拉伤 → 骨科
- ❌ 颅骨骨折 → 不归骨科

## 木桶效应

四要素能力下限决定运动水平。只有四要素均衡发展，运动系统才最高效。

# 02｜系统失衡：各种骨伤骨病从何而来？

## 失衡是根源

只有四个要素彼此平衡，运动系统才能正常运转。

## 演化层面

人类直立行走后脊柱承受压力大。35岁后身体机能下降是必然。

## 四要素层面

### 肌肉：超量修复

肌纤维损伤后会变得更粗壮。

健身时"力竭""再来一组"就是为了产生损伤，超量修复让肌肉更强。

损伤超过30%就是运动损伤。

### 骨骼：超量修复

骨折后新生骨质修复，修复后强度甚至比原来更强。

### 软骨：真脆弱

几乎无血液供应，损伤大多不可逆。

### 韧带：部分可修复

- 血液供应好的（侧副韧带）：可修复
- 无血液供应的（交叉韧带）：无法修复

# 03｜重建：骨科干预的第一思路是什么？

## 4R法则

### 1. 切除（Resection）

无法保留或保留有害的组织，一切了之。

### 2. 修复（Repair）

**缝合**：韧带、肌腱断裂
**复位**：骨折后拼接固定

### 3. 重建（Reconstruction）

用人工材料重新构造功能。

人工膝关节、人工髋关节、人工肘关节等。

### 4. 再生（Regeneration）

干细胞培养可再生组织器官。

# 04｜中枢：上班族为什么腰疼脖子疼？

## 脊柱结构

椎骨 + 椎间盘 + 韧带 + 肌肉

**椎间盘是软骨。**

## 为什么容易出问题

1. **根本原因**：直立行走让脊柱承受大压力
2. **直接原因**：久坐、长期伏案

## 病变过程

### 肌肉

长期紧绷 → 劳损

### 椎间盘

1. 被压扁 → 可恢复
2. 膨出 → 释放压力
3. 破裂 → 椎间盘突出

## 保护建议

1. 避免久坐
2. 每45分钟-1小时站起来活动
3. 加强颈腰肌肉训练（小燕飞、游泳）
4. 按摩务必找正规技师

# 05｜上肢：为什么出问题的总是关节？

## 特点：灵活但脆弱

关节活动范围大，但稳定性差。

肩关节头大窝小，像乒乓球在浅盘子上。

## 常见问题

### 脱位

肩关节脱位占全身脱位50%。

复位后必须固定6周。

### 软组织劳损

**肩周炎、网球肘、扳机指**都是过度使用导致的无菌炎症。

**与"风寒"无关**，是免疫炎症。

## 治疗

1. 限制活动
2. 消炎止痛药物
3. 耐心休息

## 保护建议

1. 让关节获得足够休息
2. 不盲目挑战极限
3. 加强上肢肌肉锻炼

# 06｜下肢：长期跑步究竟伤不伤膝盖？

## 特点：负重

下肢承担全身重量，膝关节和踝关节最薄弱。

## 膝关节：半月板损伤

突然加速、变向时受力过大导致。

**判断**：
- 剧痛 + 变形 → 可能骨折
- 能活动 + 明确痛点 → 肌腱/韧带
- 疼痛模糊 → 软骨问题

## 踝关节：崴脚

占运动损伤40%。

**判断**：
- 走路崴脚，不很疼，没肿 → 休息即可
- 跑跳扭伤，马上肿，不能站 → 去医院

**错误做法**：
- ❌ 热敷
- ❌ 坚持活动

**正确做法**：
1. 停止运动
2. 冷敷消肿
3. 休息恢复

## 保护建议

1. 减少损伤关节的活动
2. 运动佩戴护具
3. 训练股四头肌
4. 长跑要循序渐进

# 07｜代谢：日常补钙到底有没有用？

## 骨代谢

- **成骨细胞**：合成新骨骼
- **破骨细胞**：溶解旧骨骼

激素和维生素调节两者活动。

## 各阶段建议

| 年龄 | 建议 |
|------|------|
| 0-2岁 | 补充维生素D |
| 2岁后 | 均衡饮食 |
| 青春期 | 补钙 + 运动 |
| 中老年 | 补钙 + 维D + 运动 |

## 骨质疏松

**症状**：
1. 腰背疼痛，沿脊柱向两侧扩散
2. 驼背

**预防**：
1. 增加钙、维生素D、蛋白质
2. 少吸烟喝酒咖啡可乐
3. 适量运动

## 补钙

- 各种钙片效果差不多
- 重点看有没有加维生素D
- **骨头汤不补钙**

# 08｜攻防：运动系统的敌人有哪些？

## 1. 正面冲突：骨折

外力超过骨骼承受范围。

愈合：3个月拆石膏，1-1.5年彻底痊愈。

**骨折后不要长期卧床。**

## 2. 偷偷潜入：骨感染

细菌进入骨骼。

**症状**：高烧 + 骨骼关节剧痛

**治疗**：抗生素 + 钻孔减压

## 3. 内部叛乱：自身免疫疾病

### 类风湿性关节炎

免疫系统攻击关节。女:男 = 3:1。

### 强直性脊柱炎

免疫系统攻击脊柱。男:女 = 9:1。

**与"风寒湿气"无关。**

# 09｜结语：未来骨科的突破点在哪？

## 仿生医学

- 新材料接近人体
- 外骨骼增强能力

## 生物医学

- 生物制剂控制免疫疾病
- 基因编辑治疗遗传骨病

## 再生医学

干细胞培养可替换组织器官，实现永生。
