# 发刊词 | 买护肤品你到底该听谁的？

## 6个常见护肤误区

| 误区 | 真相 |
|------|------|
| 面霜吸收好才有效 | 皮肤没有吸收能力，只是挥发性成分挥发了 |
| 刺痛说明成分有效 | 刺痛是皮肤报警，不能忍 |
| 一天一张面膜好 | 一周2-3次足够 |
| 植物成分更温和 | 植物提取物成分复杂，更容易过敏 |
| 吃猪蹄补胶原蛋白 | 胶原蛋白合成有自己的规律，吃猪蹄补脂肪 |
| 每周去角质好 | 角质是保护层，频繁去角质最破坏皮肤 |

# 1. 皮肤的三大功能

## 皮肤的基本数据

- 人体**最大的器官**
- 面积接近2平方米
- 重量约占体重1/6

## 三大主要功能

### 1. 屏障功能（最重要）

**对外抵御**：
- 角质层抵御外界物质入侵
- 免疫屏障和微生物屏障抵御有害微生物
- 黑色素抵御紫外线

**对内保护**：
- 天然保湿因子锁住水分
- 失去皮肤会因脱水而死

**再生能力**：
- 皮肤是最大的干细胞库之一
- 受损后可及时恢复

### 2. 感觉功能

- 感知温度、疼痛、触觉
- 与外界的信息交流

### 3. 代谢功能

- 糖、脂肪、蛋白质代谢
- 皮脂腺、汗腺是重要的代谢通道

## 皮肤不做的两件事

1. **皮肤不呼吸** — 肺才是呼吸器官
2. **皮肤不主动吸收** — 只有有限的渗透能力

# 2. 皮肤的健康管理

## 好皮肤的四个标准

1. **亮** — 自然光泽，气色好，有血色
2. **匀** — 肤色均匀，没有明显色斑
3. **紧** — 紧实有弹性，没有太多皱纹和松弛
4. **皮实** — 耐受力好，不娇气，不易敏感过敏

**注意**：没说要白。白是社会文化评价标准，不是健康标准。

## 维护好皮肤的三件事

### 1. 养出来

- 保持良好的作息习惯和精神状态
- 激素稳定是皮肤好的基础
- 神经性皮炎主要原因就是情绪紧张

### 2. 动出来

- 运动增强体质，肌肉是皮肤的重要支撑
- 促进血液循环，清理代谢废物
- 促进胶原蛋白生成

### 3. 吃出来

- 控制高油、高糖、辛辣食物
- 控制糖摄入（水果也不能当饭吃）
- 糖与胶原蛋白发生糖化作用，导致肤色暗淡

## 千万不能做的事

**不要破坏角质层！**

### 角质层的"砖头-泥浆结构"

- 砖头：角质细胞（15层左右，厚度10-40微米）
- 泥浆：保湿因子和脂质分子

### 保护角质层

1. 不要频繁去角质
2. 避免长时间泡热水澡
3. 洗脸别洗太长时间

# 3. 皮肤的年轻管理

## 衰老两大元凶

### 1. 干燥

**危害**：
- 缺水 → 蛋白酶活性低 → 老化角质堆积
- 粗糙、干纹、没光泽 → 显老
- 严重干燥引发皲裂 → 屏障受损 → 恶性循环

**对策：保湿（不是补水）**
- 加强角质层里的脂质分子和保湿因子
- 不要让水分轻易流失

### 2. 紫外线

**危害**：
- 光老化严重程度远超自然老化
- 长波紫外线（UVA）：晒老，一年四季都要防
- 中波紫外线（UVB）：晒伤，阴天问题不大

**案例**：英国卡车司机，开车28年，左脸比右脸老20岁

## 防晒要点

1. 太阳太大尽量不硬晒
2. 物理遮挡效果最好（长袖衬衫、遮阳伞）
3. 学会看防晒指数：
   - SPF值：防暴晒能力（SPF50高倍，SPF20中等）
   - PA值：防长波紫外线（加号越多越好，最高+++）
4. 长时间户外：选高倍防晒，暴露皮肤都涂，定时补涂
5. 日常通勤：中等防晒力足够
6. 雾霾天：格外注意防晒

**适量的紫外线有好处**：调节睡眠、舒缓心情、合成维生素D

# 4. 基础护肤，三件套足够

## 安全原则

1. **三无产品绝对不用** — 没有生产商信息、生产日期、许可证号
2. **警惕三天特效美白/抗过敏产品** — 可能违法添加激素
3. **化学≠不安全，天然≠安全**
4. **不舒服的产品坚决停用**

## 基础三件套

**300块钱就能搞定**：

### 1. 清洁（洗面奶）
- 一天洗两次
- 动作温柔

### 2. 保湿（乳霜）
- 滋润保湿的产品
- 成分：甘油、丙二醇、乳酸、尿素等

### 3. 防晒（防晒霜）
- 四季常备
- 建议**两支**：中等防晒（日常）+ 高倍防晒（户外）

## 选购原则

**大牌子、老产品** — 经过时间考验的经典

# 5. 护肤升级，钱该花在哪

## 价格构成

**十分价钱三分货**：
- 一分买成分
- 一分买质地
- 一分买包装
- **七分买你高兴**（品牌溢价）

## 功能成分速查

| 功能 | 推荐成分 |
|------|----------|
| 保湿滋润 | 甘油、丙二醇、乳酸、尿素、神经酰胺、角鲨烷 |
| 美白 | 烟酰胺、熊果苷、维生素C、曲酸、光甘草定 |
| 抗老 | 视黄醇、维生素C、维生素E、胜肽、玻色因 |
| 祛痘 | 水杨酸、辛酰水杨酸、乙醇酸、烟酰胺 |

## 各类产品选购

**清洁产品**：舒服最重要

**保湿产品**：看成分，选舒服的质地

**防晒产品**：可以适当买贵一点、不油腻的

# 6. 护肤品使用秘笈

## 使用顺序：从水到油

面膜 → 护肤水 → 精华液 → 乳液 → 乳霜/面霜

## 注意事项

### 清洁
- 洗面奶不用反复洗
- 动作温柔
- 防晒霜和日常彩妆洗面奶都能洗干净

### 保湿
- 抹匀就行，不用拍打按摩
- 拍打按摩不能促进吸收

### 防晒
- 挤一元硬币大小
- 抹匀后别使劲蹭
- 出汗出油厉害要补涂

## 护肤品保存

1. **不要囤货**
2. **常温、干燥、避光** — 不要放冰箱
3. **开封后半年内用完** — 注意包装上的开封后使用期限标识（数字+M）

# 7. 医学美容初入门

## 核心原则

**做什么不是最关键的，谁给你做才是最关键的！**

## 正规机构判断标准

- 有《医疗机构执业许可证》
- 医生有执业医师资格（可在卫生部官网查询）

## 好医生的三个线索

1. 事前仔细专业诊断
2. 摆正预期，拒绝不合理要求
3. 充分告知风险，书面签字同意

## 四大主流技术

| 技术 | 原理 | 适用问题 |
|------|------|----------|
| 换肤 | 酸性溶液破坏浅表皮肤，长出新皮 | 皮肤暗沉、浅痘印 |
| 注射 | 肉毒素阻断神经肌肉信号 | 皱纹改善 |
| 激光 | 不同波长作用于血红蛋白/色素/水分子 | 祛斑、脱毛、皮肤新生 |
| 填充 | 注射填充物支撑塌陷 | 皮肤松弛、塌陷 |

## 家用美容仪

- 能量低，效果弱
- 不能和正规医美比
- 注意安全，严格按说明操作

# 8. 皮肤急救

## 敏感肌

**症状**：冷热刺激、太阳晒、迎风难受，用护肤品刺痛

**原因**：皮肤屏障受损

**处理方法**：
1. 及时停掉刺激源头
2. 及时保湿，恢复水脂平衡
3. 注意防晒
4. 症状加重看医生

## 痘痘（痤疮）

**核心认知**：痘痘是病，要看医生！

**国内皮肤科强院**：
- 北京：空军总医院
- 上海：华山医院
- 广州：中山三院
- 沈阳：中国医科大学附属第一医院
- 成都：华西医院
- 南京：南京医科大学第一附属医院

**日常管理**：
1. 治痘药物容易干燥，用乳液舒缓
2. 好好洗脸，少吃油腻少吃糖
3. 可以用粉底遮盖
4. 注意防晒

## 色斑

**成因**：黑色素异常增加，与紫外线密切相关

**处理方法**：
1. 防晒最重要
2. 护肤品祛斑效果有限
3. 可以用粉底遮盖
4. 激光效果好，但要去正规机构

---

## 划重点

1. 皮肤最重要的是屏障功能，不要破坏角质层
2. 好皮肤养出来、动出来、吃出来
3. 衰老两大元凶：干燥和紫外线
4. 基础护肤三件套：清洁、保湿、防晒
5. 医美最重要的是找正规机构、专业医生
6. 敏感要停刺激源，痘痘要看医生，色斑要防晒
