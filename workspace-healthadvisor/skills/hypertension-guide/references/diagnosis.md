# 高血压诊断与分级

## 诊断标准（因年龄而异）

| 年龄段 | 满意血压 | 说明 |
|--------|----------|------|
| 25-35岁 | 115/75 mmHg | 为衰老留出空间 |
| 40岁前 | <120/80 mmHg | 预防黄金期 |
| 40-50岁 | 125/80 mmHg | 关注血压变化 |
| 55岁前 | <135/85 mmHg | 可接受范围 |
| 65岁+ | <140/90 mmHg | 考虑供血需求 |

**基础血压原则**：基础血压升高，收缩压超过20mmHg、舒张压超过10mmHg，也是判断高血压的重要指标。

## 高血压分级（诊室血压）

```
正常血压：    < 120/80 mmHg
正常高值：    120-139 / 80-89 mmHg  ← 预防窗口期
1级高血压：   140-159 / 90-99 mmHg  ← 逆转黄金期
2级高血压：   160-179 / 100-109 mmHg
3级高血压：   ≥ 180/110 mmHg
```

**注意**：收缩压和舒张压分属不同级别时，以较高的级别为准。

## 测量方法

### 诊室血压
- 四周内测2-3次，每次都达到140/90mmHg才可确诊
- 注意"白大衣高血压"（见医生紧张）和"隐匿性高血压"

### 家庭测量
- 连续7天，早晚各测2次，间隔1分钟
- 剔除第一天，取平均值
- ≥135/85mmHg可诊断为高血压

### 24小时动态血压
- 全天平均 ≥130/80mmHg
- 白天平均 ≥135/85mmHg
- 夜间平均 ≥120/70mmHg
- 任一达标即为高血压

## 家庭测量要点

1. 测量前半小时不抽烟、喝酒、咖啡，安静3-5分钟
2. 安静房间，坐有靠背的椅子，双脚平放
3. 袖带绑在肘部以上，松紧能容一指
4. 肘部、心脏、血压计同一水平
5. 选择有中国药监局认证的上臂式电子血压计
