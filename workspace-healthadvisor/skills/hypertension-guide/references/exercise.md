# 高血压运动处方

## FITT原则

| 要素 | 建议 |
|------|------|
| F（频次） | 每周5-7次 |
| I（强度） | 中等强度（最大摄氧量40-60%） |
| T（类型） | 有氧运动为主 |
| T（时间） | 每次30分钟（不含热身整理5+5分钟） |

## 强度计算

### 中等强度心率公式
```
最佳心率 = (220 - 年龄 - 静息心率) × 40%~60% + 静息心率
```

### 简易判断
- 运动时能说话但不能唱歌
- 微微出汗，呼吸加快但不气喘

**注意**：血压超过160/100mmHg或有心血管疾病、糖尿病者，需在医生指导下确定强度。

## 运动类型效果排序

### 1. 等长力量训练（效果最佳）

**典型动作**：
- 平板支撑
- 靠墙静蹲
- 马步
- 踮脚端水盆

**降压效果**：月降收缩压10.9mmHg，舒张压6.2mmHg

**执行要点**：
- 从1分钟开始，隔天进行
- 逐步增加时间
- 注意避免憋气

**风险提示**：可能导致血压瞬间过高，建议先在医学监护下尝试。

### 2. 高强度间歇训练

**典型方案**：
- 高强度（85-95%最大摄氧量）3分钟
- 低强度（50%最大摄氧量）3分钟
- 交替循环4组

**优点**：效率高，改善血管弹性

**注意**：关节肌肉损伤者需咨询医生

### 3. 中等强度有氧运动

**典型运动**：跑步、游泳、骑车、划船机

**降压效果**：
- 高血压前期：收缩压降4.3mmHg，舒张压降1.7mmHg
- 确诊高血压：收缩压降8.3mmHg，舒张压降5.2mmHg

### 4. 一般力量训练

**典型动作**：哑铃、推举、引体向上、俯卧撑

**执行要点**：
- 每周3次，隔天进行
- 每次8个动作，每个10次

**降压效果**：收缩压约降3mmHg

## 综合方案示例

1. 热身5分钟
2. 中等强度跑步30分钟（每周5天）
3. 力量训练8个动作×10次（每周3天）
4. 平板支撑2分钟（每周3天）
5. 拉伸5分钟

## 注意事项

- 规律运动3个月后可改为每周3-5次
- 完全停掉会导致血压反弹
- 运动前后监测血压
