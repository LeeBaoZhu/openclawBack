# 高血压饮食处方

## 方案一：减盐饮食

### 目标
每天盐摄入 **3.5-6g**

### 执行要点

**1. 使用小盐勺**
- 知道一勺是几克
- 逐渐换小盐勺

**2. 警惕隐形盐**

| 类别 | 常见食物 |
|------|----------|
| 调味品 | 酱油、鸡精、味精、蚝油、豆瓣酱、甜面酱、腐乳 |
| 精加工熟食 | 香肠、腊肉、熏肉、豆腐干、汉堡、方便面、挂面 |
| 甜食 | 雪糕、蛋糕、冰淇淋、奶茶、蛋挞、话梅 |

**3. 多吃富钾食物**
- 新鲜蔬菜、水果、坚果、豆类
- 例：两个橘子含钾约0.5g

**4. 替代口感**
- 用醋、辣椒、花椒、大料、胡椒
- 推荐辣椒（辣椒素对心脑血管有保护作用）
- 多加西红柿、香菜、葱、姜、蒜

**5. 看食品标签**
- 选择钠 <0.12g/100g 的食物
- 超过0.4g钠/100g 的不建议选择

**效果**：坚持4周，血压下降4-6mmHg

## 方案二：DASH饮食

### 核心原则
以全谷物为基础，科学配比

### 每日摄入参考（2000大卡/天）

| 类别 | 份量 | 每份示例 |
|------|------|----------|
| 全谷物 | 6-8份 | 半碗糙米饭/1片全麦面包 |
| 蔬菜 | 4-5份 | 半碗熟菜 |
| 水果 | 4-5份 | 1个中等水果（苹果、梨、橙子、香蕉） |
| 肉类 | 5-6份 | 30g鱼肉/虾肉（去皮去肥肉） |
| 奶制品 | 2-3份 | 1杯低脂奶(250ml) |
| 油脂 | 2-3份 | 1汤勺 |
| 坚果 | 3-5份/周 | 2汤勺（无盐无糖不油炸） |

### 四格餐盘法
- 两格菜
- 一格碳水
- 一格肉

**效果**：单独使用可降低收缩压11mmHg
