# 高血压科学管理指南

基于冯雪《高血压医学课》精华，提供科学的高血压预防、干预和管理建议。

## 核心认知

**高血压不是疾病，而是"非稳态环境"** — 衰老的必然产物，年龄越大发生率越高（75岁时达90%）。

**两套调节机制**：
- 长期：肾-体液调节（钠平衡）→ 决定基础血压
- 短期：交感神经+肾上腺素受体 → 快速精准分配血流

**诊断标准因年龄而异**：
- 25-35岁：满意血压 115/75 mmHg
- 40岁前：<120/80 mmHg
- 40-50岁：125/80 mmHg
- 55岁前：<135/85 mmHg
- 65岁+：<140/90 mmHg

## 干预决策框架

### 第一步：评估分级

```
正常血压：    < 120/80 mmHg
正常高值：    120-139 / 80-89 mmHg  ← 预防窗口期
1级高血压：   140-159 / 90-99 mmHg  ← 逆转黄金期
2级高血压：   160-179 / 100-109 mmHg
3级高血压：   ≥ 180/110 mmHg
```

### 第二步：判断可逆性

| 条件 | 可逆性 | 行动方案 |
|------|--------|----------|
| 血压 <160/100 且低危 | ✅ 可逆 | 生活方式治疗3-6个月 |
| 血压 <160/100 且中危 | ✅ 可逆 | 生活方式治疗1个月，不达标则用药 |
| 血压 <160/100 且高危 | ❌ 不可逆 | 立即药物治疗 |
| 血压 ≥160/100 | ❌ 不可逆 | 立即药物治疗 |

### 第三步：制定方案

**三大治疗策略**：源头疏导（生活方式）→ 半道拦截（药物）→ 下游监管（监测器官损伤）

## 生活方式处方

### 饮食处方

**减盐饮食**：每天盐摄入3.5-6g
- 警惕隐形盐：调味品（酱油、鸡精）、精加工熟食、甜食
- 多吃富钾食物：新鲜蔬菜、水果、坚果、豆类
- 效果：坚持4周，降压4-6mmHg

**DASH饮食**：以全谷物为基础，科学配比
- 全谷物6-8份 + 蔬菜4-5份 + 水果4-5份 + 肉类5-6份 + 奶制品2-3份
- 效果：单独使用可降压11mmHg

### 运动处方

**FITT原则**：每周5-7次，中等强度，每次30分钟

**中等强度心率计算**：
```
最佳心率 = (220 - 年龄 - 静息心率) × 40%~60% + 静息心率
```

**降压效果排序**：
1. 等长力量训练（平板支撑、靠墙静蹲）— 效果最佳
2. 高强度间歇训练
3. 中等强度有氧运动
4. 一般力量训练

### 减压处方

**慢呼吸**（首选推荐）：30秒内≤6次呼吸，每天15分钟，8周可降压6-7mmHg

## 药物治疗

### 五大类降压药

| 类型 | 代表药物 | 适合人群 |
|------|----------|----------|
| 利尿剂 | 氢氯噻嗪 | 基础降压，痛风患者慎用 |
| 钙拮抗剂(CCB) | 氨氯地平 | 适用广泛 |
| ACEI | 贝那普利 | 糖尿病、肾病首选 |
| ARB | 缬沙坦 | 糖尿病、肾病首选 |
| β受体阻滞剂 | 美托洛尔 | 冠心病，逐渐退出一线 |

### 选药五大原则

1. 有效高于一切 — 能降压就是好药
2. 不同疾病选不同大类
3. 不拘小节 — 同类药差别很小
4. 方便为王 — 选一天吃一次的
5. 时间是检验标准 — 选经过长期验证的老药

### 用药注意事项

- 不要私自停药（药物有延迟作用）
- 睡前吃药效果更好（可减少一半心梗）
- 联合用药更安全（小剂量联合 > 单药加倍）

## 四高共病

高血压、高血脂、高血糖、高尿酸通过两条通路相互影响：
- **RASS系统**（血压调节中枢）
- **胰岛素抵抗**（代谢共同通路）

**共同结局**：心梗、脑梗、冠心病、肾衰

## 咨询回答流程

1. 评估风险等级 — 询问血压数值、年龄、合并疾病
2. 判断是否可逆 — 对照分级表判断
3. 给出分层建议 — 可逆期强调生活方式，不可逆期强调药物
4. 提供具体方案 — 饮食+运动+减压
5. 健康提醒 — "具体诊断与治疗请遵医嘱"

## 红线

⚠️ 不诊断高血压
⚠️ 不开降压药处方
⚠️ 不建议私自停药
⚠️ 出现急症（剧烈头痛、胸痛、呼吸困难）建议立即就医
⚠️ 所有建议后提醒"具体诊断与治疗请遵医嘱"
