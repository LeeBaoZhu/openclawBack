# 发刊词｜你能从这门课听到什么

## 健身的目的

让你在高强度工作中游刃有余，工作之外还有精力享受生活。

## 常见问题

很多人大量训练后更疲劳，更睡不着觉。

## 健康身体五要素

1. 心肺功能（最重要）
2. 脂肪
3. 柔韧度
4. 肌肉耐力
5. 肌肉力量

**肌肉不是健身的首要目的。**

# 1. 心肺功能是一切的基础

## 心肺系统的重要性

心血管系统好比汽车发动机，是最重要的。

**风险**：心肺功能一般的人做高强度运动可能猝死。

## 自测：最大摄氧量

最大摄氧量 = 运动中能获取的最大氧气量

| 水平 | 男性 | 女性 |
|------|------|------|
| 及格 | ≥40 | ≥36 |
| 优秀 | ≥54 | — |
| 顶级运动员 | 可达88 | — |

**低于及格线有猝死风险。**

## 自测方法

买专业心率表（推荐），或用手测脉搏。

## 运动心率计算

**卡氏公式**：
```
心肺训练心率 = (220 - 年龄 - 静态心率) × (55%~65%) + 静态心率
```

## 训练方法

**推荐**：跑步机上坡走

- 速度：4-6公里/小时
- 坡度：根据心率调整
- 原因：不双脚腾空，膝盖压力小

## 注意事项

- 不要跑太快，速度4-6刚好
- 训练一天休息一天
- 睡眠不好时优先补觉，不要高强度训练

# 2. 如何科学有效地减肥

## 三个错误方法

### 1. 增肌减肥

**问题**：效率极低

- 肌肉只占基础代谢20%
- 每增加1kg肌肉，基础代谢只多50卡
- 增1kg肌肉需狂练一个月

### 2. 高强度运动减肥

**问题**：消耗的是糖和水，不是脂肪

- 脂肪消耗需要氧气参与
- 上气不接下气时氧气不足
- 体重减轻是糖和水

### 3. 节食减肥

**问题**：必反弹，免疫力下降不可逆

- 食欲无法用意志力对抗
- 节食后免疫力下降，补不回来

## 正确减脂方法

### 减脂心率

卡氏公式的35%~55%（比心肺训练强度更低）

### 关键：饮食控制

**糖、蛋白质、脂肪吸收率**：
- 糖：转化率约70%
- 脂肪：转化率约96%
- 蛋白质：只吸收需要的量

### 减脂饮食公式（25岁以上）

| 营养素 | 摄入量 |
|--------|--------|
| 碳水 | 体重 × 1.8g（女）或 2g（男） |
| 脂肪 | 体重 × 1g |
| 蛋白质 | 体重 × 1.4g |

**示例**（60kg男性）：
- 碳水：120g
- 脂肪：60g
- 蛋白质：84g

### 注意

- 一个中等红薯 ≈ 两碗米饭
- 肉和炒菜里脂肪已够30%

# 3. 柔韧度不好真的要命

## 柔韧度的重要性

对生活质量影响比肌肉大得多：
- 梳头、挠后背困难
- 手臂抬不过肩膀
- 关节僵硬疼痛

## 自测

坐姿体前屈：
- 能碰到脚尖 → 还可以
- 只能碰到膝盖 → 需要改善

## 两种拉伸

| 类型 | 方法 | 时机 |
|------|------|------|
| 静态拉伸 | 保持15-30秒 | 运动后 |
| 动态拉伸 | 连续动作 | 运动前 |

**运动前做静态拉伸会降低肌肉力量！**

## 正确运动顺序

```
1. 泡沫轴放松
2. 动态拉伸（10-20分钟）
3. 专项训练
4. 静态拉伸
```

# 4. 肌肉耐力让你享受运动

## 肌肉三个指标

1. 肌肉耐力
2. 肌肉力量
3. 肌肉量

## 肌肉耐力自测

徒手深蹲30次：
- 能完成 → 正常
- 做不到 → 需要改善

## 训练方法

**走路就能改善！**

**跑步机上坡走技巧**：
- 站在跑步机中央
- 肚脐指向前方
- 加大步幅和摆臂
- 脚尖始终向前

# 5. 怎样应对突发状况

## 肌肉力量自测

| 动作 | 标准 |
|------|------|
| 跪姿俯卧撑 | 能完成 |
| 单杠拉起 | 能拉起60%体重 |
| 负重深蹲 | 负重=体重做1次 |

## 训练要点

**每组必须做到力竭** — 再也做不动为止。

## 外形改善

### 腹肌

前提：体脂足够低
核心：控制饮食减脂

### 翘臀

- 上坡走能练到臀大肌
- 壶铃摇摆：每周3天，每天70次

### 小腿粗

原因：走路姿势错误
解决：泡沫轴放松 + 改正姿势

## 抗衰老肌肉

- 臀大肌
- 股四头肌
- 腹肌
- 背肌
