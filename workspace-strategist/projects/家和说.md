# 项目：家和说

## 产品概述

**产品名**：家和说
**定位**：以「家庭」为单位的轻量级互动软件
**一句话**：家庭的私密社交空间，让家人之间能吐槽、赞扬、期待

## 核心价值

- 改善家庭沟通
- 促进家庭成员理解
- 为家庭提供私密的虚拟空间

## 功能规划

### MVP版本（已完成）
- [x] 发布反馈
- [x] 反馈列表
- [x] 家庭管理

### 第一阶段：核心功能
- [ ] 家庭创建（邀请码加入）
- [ ] 发布内容：吐槽/赞扬/期待
- [ ] 可见范围控制
- [ ] 基础展示

### 第二阶段：扩展功能
- [ ] 周会讨论
- [ ] 行动计划
- [ ] 积分激励
- [ ] 家庭日历
- [ ] 纪念日提醒
