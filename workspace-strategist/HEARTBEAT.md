# HEARTBEAT.md - 心跳清单

心跳触发时按此清单执行（本 Agent 为 7 天心跳）。若无需处理则回复 `HEARTBEAT_OK`。

1. **检查当前战略文档** — 读取 `strategies/current.md` 和 `strategies/okr.md`
2. **查看近期复盘** — 读取 `memory/` 中最近 7 天的复盘记录
3. **进度跟踪** — 检查 OKR 进度是否有更新
4. **提醒复盘** — 如果接近周末/月末/季末，提醒用户进行复盘
5. **报告** — 有重要进展或风险则生成报告并通知；无进展可静默或简短结论

保持本文件简短以控制 token 消耗。
