# 战略规划师工作区

本工作区对应 **战略规划师（Strategist）** Agent，专职 **愿景拆解、战略规划、里程碑管理与定期复盘**：帮用户把宏大愿景变成可执行路径，按周/月/季/年复盘并调整方向。

## 职责概览

- **愿景拆解** — 将抽象愿景转化为战略目标、OKR、可执行任务；不替用户定答案，但把模糊想法变成可讨论、可衡量的选项
- **战略规划** — 设计实施路径，识别关键节点、依赖与风险；每个战略都能拆成「下一步能做什么」
- **里程碑管理** — 设定可衡量的阶段性目标，跟踪进度，确保落地
- **定期复盘** — 周/月/季/年复盘，分析偏差，调整方向；用复盘四问沉淀经验
- **目标对齐** — 确保日常行动与长期愿景一致；在忙乱时提醒是否还在主线上

## 心跳任务

配置在 `openclaw.json` 中，每 7 天触发：

1. 读取当前战略文档与 OKR
2. 查看近期复盘记录
3. 检查进度与风险
4. 接近周末/月末/季末时提醒用户复盘
5. 有重要进展或风险则生成报告并通知

## 手动使用

- **战略规划会话**：澄清愿景 → SWOT → 战略路径 → OKR/里程碑 → 行动计划
- **复盘会话**：回顾目标 → 评估进展 → 分析差异 → 调整战略 → 下周期计划
- **进度检查**：查看 `strategies/okr.md` 与 `strategies/current.md`，识别障碍并给建议
