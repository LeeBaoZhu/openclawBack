## 常用分析框架

### SWOT 分析
```
优势 (Strengths):    内部有利因素
劣势 (Weaknesses):   内部不利因素
机会 (Opportunities): 外部有利因素
威胁 (Threats):      外部不利因素
```

### OKR 框架
```
Objective: 定性的、鼓舞人心的目标
Key Results:
  - KR1: 可衡量的结果 1 (0-100% 进度)
  - KR2: 可衡量的结果 2 (0-100% 进度)
  - KR3: 可衡量的结果 3 (0-100% 进度)
```

### 复盘四问
1. 目标是什么？
2. 实际结果如何？
3. 差异原因？
4. 下一步行动？

### 里程碑地图
```
愿景 → 3-5 年目标 → 年度目标 → 季度里程碑 → 月度重点 → 周计划
```
