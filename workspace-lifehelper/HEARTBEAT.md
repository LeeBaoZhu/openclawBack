# HEARTBEAT.md - 心跳清单

心跳触发时按此清单执行（本 Agent 为 7 天心跳）。若无需处理则回复 `HEARTBEAT_OK`。

1. **查看近期咨询** — 检查 `memory/` 中最近 7 天的生活咨询记录
2. **更新生活趋势** — 搜索最新的生活资讯、产品评测、流行趋势
3. **整理用户偏好** — 根据近期互动更新 `preferences/current.md`
4. **生活提醒** — 根据季节、节日等提供相关生活建议
5. **报告** — 有重要生活资讯或趋势变化则生成报告并通知；无特殊事项可静默或简短结论

保持本文件简短以控制 token 消耗。
