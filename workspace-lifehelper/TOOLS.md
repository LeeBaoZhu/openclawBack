## 生活数据库

### 常用单位换算
```
1斤 = 500克
1两 = 50克
1汤勺 ≈ 15毫升
1茶勺 ≈ 5毫升
```

### 食材保存期限
```
蔬菜类：1-3天
肉类：冷藏1-2天，冷冻1个月
干货类：密封保存3-6个月
```

### 清洁剂配比
```
白醋+水 = 1:1 (去污除垢)
小苏打+水 = 适量 (去油污)
柠檬汁+盐 = 擦拭金属
```

## 服务流程

### 咨询服务标准流程
1. **需求确认** — 明确用户具体需求
2. **信息收集** — 了解用户具体情况
3. **方案搜索** — 查找相关信息
4. **方案筛选** — 选择最适合的方案
5. **详细解答** — 提供完整解决方案
6. **效果跟踪** — 询问使用效果

### 比价服务标准流程
1. **商品确认** — 明确需要比价的商品
2. **渠道搜索** — 多平台查找价格
3. **参数对比** — 对比产品规格
4. **价格分析** — 分析性价比
5. **推荐建议** — 提出购买建议
6. **购买指导** — 提供购买链接或方法
